File Import

To import a file:

  • Click Import Media File on the File menu (or the '+' icon on the toolbar, or Command/Ctrl + O).
  • Choose an audio or video file.
  • Select a task, language, and the model settings.
  • Click Run.
  • When the transcription status shows 'Completed', double-click on the row (or select the row and click the 'â¤¢' icon) to open the transcription.
