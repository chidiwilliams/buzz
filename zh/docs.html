介绍

在您的个人电脑上离线转录和翻译音频。由 OpenAI 的 Whisper 提供支持。

功能​

  • 导入音频和视频文件，并将转录内容导出为 TXT、SRT 和 VTT 格式（演示）
  • 从电脑麦克风转录和翻译为文本（资源密集型，可能无法实时完成，演示）
  • 支持 Whisper、 Whisper.cpp、Faster Whisper、 Whisper 兼容的 Hugging Face 模型 和 OpenAI Whisper API
  • 命令行界面
  • 支持 Mac、Windows 和 Linux
